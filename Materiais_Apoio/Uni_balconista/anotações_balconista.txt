	Balconista

vendas são realizadas a todo momento pelo balconista]

vendas modulo movimento 

você pesquisa os produtos pelo seu código, nome do produto e código de barras

barra de pesquisa na tela venda: limpar, eliminar, pbm's, opções, movimentação 
de estoque, cliente, produto, devolução e digitalizar
-------------------------------------------------------------------------------

geralmente na tela de venda vc define a quantidade vendida e o desconto do 
produto

pressione enter para atualizar a os dados da venda 

operações de pagamento: dinheiro, cheque, cartão e prazo

preencha os dados de cadastro do cliente em uma venda à vista caso a farmacia 
deseje

para uma venda à prazo será obrigatorio para um controle de venda e de histori
co do cliente 
------------------------------------------------------------------------------- 

vendas no cartão débito tbm não precisam de vincular o cliente

para vendas no cartão vc irá preencher os campos: operadora do cartão, tipo de 
pagamento, NSU (Número Sequencial Único), quantas parcelas e o valor da venda

clique na barra de opções, na opção de nova venda para criar uma nova venda 
sem interferir na anterior
------------------------------------------------------------------------------- 

caso houver um desconto máximo do produto, ele aparecerá na tela e a sua procen
tagem

o desconto maximo pode ser criado por grupos de medicamentos

o desconto maximo dos genericos é 40% e dos eticos é 10%
------------------------------------------------------------------------------- 

venda pendente, o cliente compra no balcão e paga no caixa

vendas no modulo movimento vá em opções na barra de operações e selecione a 
opção vendas pendentes

vendas no modulo movimento vá em opções na barra de operações e selecione a 
opção ultimas vendas para pesquisar vendas anteriores 
------------------------------------------------------------------------------- 

farmacia popular transaction centre, vidalink, e-pharma, orizon, pharmalink,
funcional card e pec febrafar

na barra de opções clique em pbm e selecione o pbm desejado
------------------------------------------------------------------------------- 

farmacia popular é o sistema de desconto oferecido pelo governo 

para medicamentos que tratam: asma, hipertenção e diabetes a compra acaba sendo
de graça

adiciona os campos: qtde ao dia, preço, total venda e apresentação, são os 
campos obrigatorios para venda do farmamcia popular

vendas no farmacia popular não precisam necessariamente do cadastro de cliente
vinculado na compra 

é solicitado o CPF do cliente o CRM do prescritor e a UF

prescrição é a data que a receita foi emitida

após preencher os campos o sistema fara uma comunicação com o farmacia popular
para liberar ou não a compra para o cliente
------------------------------------------------------------------------------- 

para realizar a venda no pbm transaction centre o sistema precisa estar configu
rado para realizar uma venda 

para uma venda direto no inovafarma clique na opção normal, caso faça pelo sis
tema do transaction centre ele ira gerar uma autorização para o inofarma

vincular o cadastro do cliente ou não na venda é opcional

o valor de desconto será mostrado na tela que define se a compra foi liberada 
ou não
------------------------------------------------------------------------------- 

para realizar a venda no pbm e-pharma o sistema precisa estar configurado para
realizar uma venda

vincular o cadastro do cliente ou não na venda é opcional

o app do inovafarma solicita uma abertura do e-pharma e logo depois abrira a 
lista de informações

convenio do cliente, numero do cartão, data da receita 

no campo imagem receita o e-pharma solicita a digitalização da receita, caso 
seja um produto controlado
-------------------------------------------------------------------------------  

para realizar a venda no pbm funcional card o sistema precisa estar configurado
para realizar uma venda

vincular o cadastro do cliente ou não na venda é opcional

na tela de comunicação com o PBM insira os campos: numero do cartão, data da
receita e os campos do prescritor
-------------------------------------------------------------------------------  
para realizar a venda no pbm VidaLink o sistema precisa estar configurado
para realizar uma venda

as vendas pelo VidaLink, são direto do aplicativo deles, ele apenas gera uma 
autorização para o inovafarma

após preencher a venda no app vc pega a autorização e joga no inovafarma

vincular o cadastro do cliente ou não na venda é opcional
------------------------------------------------------------------------------- 

para realizar a venda no pbm Orizon o sistema precisa estar configurado
para realizar uma venda

as vendas pelo Orizon, são direto do aplicativo deles, ele apenas gera uma 
autorização para o inovafarma

após preencher a venda no app vc pega a autorização e joga no inovafarma

vincular o cadastro do cliente ou não na venda é opcional
------------------------------------------------------------------------------- 

para realizar a venda no pbm PharmaLink o sistema precisa estar configurado
para realizar uma venda

vincular o cadastro do cliente ou não na venda é opcional

na tela de comunicação com o PBM insira os campos: numero do cartão, projeto,
nmr crm e uf
------------------------------------------------------------------------------- 

para realizar a venda no pbm PEC Febrafar o sistema precisa estar configurado
para realizar uma venda

neste PBM é obrigatorio na tela de venda vincular o nome do cliente e o nome 
do vendedor 

no momento após avançar na venda o servidor do inovafarma irá se comunicar com
o servidor da febrafar para concretizar na venda

o vinculo do cliente é vinculado automaticamente na tela de vendas 
------------------------------------------------------------------------------- 
 
a função digitalizar aparece em todos os estagios da venda

as funções para digitalizar são: digitalizar por uma impresssora, digitalizar 
por um aplicativo ou procurar uma imagem 
------------------------------------------------------------------------------- 

faltas e encomendas no modulo movimento

neste tela aparecerão as opções: filtar, incluir, alterar, vender, eliminar, 
status, produto, imprimir e atualizar
