automaticamente na tela

uma matriz e filial podera tranmitir medicamentos controlados entre uma e outra 

identificar c o CNPJ é da mesma matriz ou filial, se os numeros antes da barra forem iguais e os numeros depois
da barra forem diferentes 

so é possivel fazer a transferencia de medicamentos se as empresas possuirem a mesma base de CNPJ   

quando uma filial remaneja o produto para a outra quer dizer que ela vai enviar produtos de uma loja para a outra

ao enviar um remanejo mesmo que no sistema a operação esteja como tranferencia no sistema do SNGPC vai estar como
uma saida de estoque e para quem recebeu será uma compra de estoque
